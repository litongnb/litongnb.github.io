JAVA基础

数组

数组概念：

数组是一种容器，可以用来春促同种数据类型的多个值
存储数据时需要结合隐式转换考虑

如果定义int数组，double,boolen这些类型的数组时不可以存入的，
但是，byte,shrot,char这些类型因为会自动转换可以存入

建议：容器的类，和存储的数据类型保持一致。

数组定义：

格式一：

数据类型 [] 数组名

比如：int [] arr 数据类型：int 数组名：arr

格式二：

数据类型 数组名 []

比如： int arr []

数组静态初始化

完整格式：

数据类型 [] 数组名 = new 数据类型[]{元素1，元素2，元素3…};

比如：
int [] arr = new int[]{1 , 2 , 3};

​ double [] arr = new double[]{1.1 , 2.2 , 3.3};

格式详解：

数据类型： 限定数组以后可以存放什么类型数据

方括号： 表示现在定义的是一个数组

数组名： 给数组命名

注意点：

等号前后数据类型一致，创建完成数组后，长度不得发生改变

简化格式：

数据类型 [] 数组名 = {元素1，元素2，元素3…};

练习1：

定义存储3个学生的姓名

练习2：

定义存储5个学生的年龄

练习3：

定义存储4个学生的身高

地址值

数组的地址值：表示数组在内存中的位置

[ ：表示现在打印的是一个数组。
I：表示现在打印的数组是int类型的。
@：仅仅是一个间隔符号而已。
6d03e736：就是数组在内存中真正的地址值。（十六进制的）
但是，我们习惯性会把[I@6d03e736这个整体称之为数组的地址值。

数组元素访问

格式：

数组名 [索引]

作用：

获取数组中对应索引上的值
修改数组中对应索引上的值
一旦修改之后，原索引上的值就会被覆盖

代码示例：

索引

也叫角标、下标

就是数组容器中每一个小格子对应的编号。

相当于一个标有序号的关键字链

索引的特点
  1. 索引一定是从0开始的
  2. 索引不间断
  3. 索引依次增加1

数组的遍历

数组的动态初始化

格式：

数据类型 [] 数组名 = new 数据类型[数组的长度]

举例：
数组的默认初始化：

整数类型：0

小数类型：0.0

布尔类型：false

字符类型：’\u0000’

引用类型：null

数组两种初始化方式的区别

静态初始化：int [] arr = {1,2,3,4,5};

动态初始化：int [] arr = new int [3];

静态初始化：手动指定数组的元素，系统会根据元素的个数，计算出数组的长度。

动态初始化：手动指定数组长度，由系统给出默认初始化值。

使用场景：

只明确元素个数，但是不明确具体的数据，推荐使用动态初始化。

已经明确了要操作的所有数据，推荐使用静态初始化。

举例：

录入学生成绩，也已知学生成绩

就用int [] arr = { };存放

如果只知道要录入多少，就动态初始化，给出一个数组的长度

如 int [] arr = new int[5];

数组常见问题

当访问了数组中不存在的索引，就会引发索引越界异常

避免：针对于任意一个数组，索引的范围，最小索引：0 最大索引：数组的长度 - 1 也就是 数组名.length - 1
